# Design Notes

This directory is reserved for design documents produced after the exploratory shape converges.

The first expected design artifact should define:

- the initial public module topology for `@beep/semantic-web`
- the boundary between pure schemas, adapters, and service contracts
- the ownership and migration plan for `IRI` and `ProvO`
- the minimum first-class JSON-LD surface
- the semantic schema metadata annotation pattern
