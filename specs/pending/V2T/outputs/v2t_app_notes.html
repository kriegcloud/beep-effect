01 — product overview
What this product is

V2T is an agentic media production platform that transforms recorded conversations into rich, animated video content. The user records a conversation — personal, professional, or exploratory — and the system handles everything from raw audio to a publishable animation.

The product targets creators, knowledge workers, and professionals who want to document and redistribute their conversations in an engaging visual format without a full video production workflow. Think of it as a conversation-to-content pipeline with a creative layer on top.

The core value proposition: your conversations already contain interesting ideas. V2T makes those ideas shareable without requiring you to be a video editor, voice actor, or animator.

Technical identity: a local-first agentic system using ElevenLabs for transcription/speech, Qwen2-Audio-7B for audio embeddings, Graphiti + FalkorDB as the knowledge graph/memory layer, and Grok-based video generation for animation output.

02 — core user experience
How a user moves through the product

Record. The user initiates a recording session — casual coffee chat, on-site discovery meeting, professional interview, or solo voice memo. They can attach optional notes or instructions before or after: context about who they talked to, what topics they expected, what kind of output they want.

Wait & review. The system automatically transcribes, identifies speakers, tags topics, cross-references memory from prior conversations, and enriches references with light web research. The user receives a structured Obsidian-compatible markdown document — a shareable, navigable artifact of the conversation, not just a raw transcript.

Configure output. The user selects an animation style, aesthetic, and target format. They choose whether the animation uses only the raw captured audio for dubbing, or whether the system generates enhanced dialogue to fill narrative gaps and improve pacing. They also specify output length: full feature, or a set of short-form clips optimized for social channels.

Generate & distribute. The composition agent produces the animation. The user can export the full video or individual clips, ready to post directly to social media without further editing.

03 — module breakdown
System modules and responsibilities
Audio Recorder
Captures conversation audio and stores in .mp3 format. Maintains SQLite database of all recordings and transcripts with a dedicated manager interface.
mp3sqlitelocal disk
Transcription Engine
Uses ElevenLabs to produce a structured markdown transcript with speaker didentification, timestamps, and metadata. Saves to filesystem and SQLite. Obsidian manifest compatible.
elevenlabsmarkdownobsidian
Analysis Agent
Processes user notes and the transcript to produce contextual documentation: people identified, environment/ambiance, topics explored, and light web-enriched context to fill gaps or connect to prior conversations.
agenticweb researchmemory xref
Audio Indexer
Uses Qwen2-Audio-7B locally to generate audio embeddings from the recording. These embeddings feed the Composition Agent for semantically-aware clip selection and scene matching.
qwen2-audio-7bembeddingslocal inference
Knowledge Base
Graphiti-powered memory layer with FalkorDB for semantic search and graph retrieval. Enables cross-conversation memory so the agent can reference context from past recordings and relationships.
graphitifalkordbsemantic search
Composition Agent
Core generation agent. User selects a generation target and triggers a run. Agent uses the knowledge base, transcript, embeddings, and style config to produce Grok video generation output — full feature or clip series.
grok video genagentic toolsclip management
Style & Dialogue Controller
User-facing controls for animation aesthetic, visual style, and audio mode. Toggle between raw audio dubbing only vs. AI-enhanced dialogue generation. Controls output format for full-length vs. social clips.

Export & Clip Manager
Takes generated animation output and produces final deliverables: full-length video, plus a series of short-form clips sized and formatted for social media distribution.
