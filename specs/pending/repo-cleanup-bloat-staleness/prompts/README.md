# Prompt Assets

These prompt assets are reusable helpers for tightly scoped cleanup sessions.

## Prompt Set

- [CANDIDATE_EXECUTOR_PROMPT.md](./CANDIDATE_EXECUTOR_PROMPT.md) — execute one approved P4 cleanup candidate in a fresh scoped session
- [FINAL_VALIDATOR_PROMPT.md](./FINAL_VALIDATOR_PROMPT.md) — run final validation and closeout review

## Prompt Rule

These assets stay generic to the cleanup workflow. Phase-specific scope belongs in the handoffs and phase outputs, not inside these reusable prompts.
