# Handoffs

This directory preserves the active phase handoffs and orchestrator prompts for the repo cleanup spec.

## Usage Guidance

- Read `AGENTS.md`.
- Start with the root [README.md](../README.md).
- Check the active phase in [outputs/manifest.json](../outputs/manifest.json).
- Open the matching handoff.
- Run the matching orchestrator prompt.
- Write or refine the named phase output in `../outputs/`.
- Update the shared checklist, manifest, and grill log when the phase requires it.
