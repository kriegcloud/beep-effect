# Handoffs

This directory preserves the active phase handoffs and orchestrator prompts for the agent governance control plane.

## Usage Guidance

- Start with the root [README.md](../README.md).
- Open the handoff for the active phase.
- Run the matching orchestrator prompt.
- Write or refine the named phase output in `../outputs/`.
- Update `../outputs/manifest.json` when the phase closes.
