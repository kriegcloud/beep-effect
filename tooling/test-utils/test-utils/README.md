# @beep/test-utils

An internal library containing shared testing utilities

## Installation

```bash
bun add @beep/test-utils
```

## Usage

```ts
import { VERSION } from "@beep/test-utils"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
