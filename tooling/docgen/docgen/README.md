# @beep/docgen

An opinionated documentation generator for beep-effect

## Installation

```bash
bun add @beep/docgen
```

## Usage

```ts
import { VERSION } from "@beep/docgen"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
