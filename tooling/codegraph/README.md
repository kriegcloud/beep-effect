# @beep/code-graph

An library for generating knowledge graphs for typescript repos

## Installation

```bash
bun add @beep/code-graph
```

## Usage

```ts
import { VERSION } from "@beep/code-graph"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
