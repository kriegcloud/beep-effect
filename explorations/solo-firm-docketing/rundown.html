Operating constraints

These facts shape the architecture more than the UI does.

ODP Sequential poller

USPTO ODP blocks concurrent same-key calls. Poll one-at-a-time per API key.

CourtListener Webhooks over polling

Free REST limits are too tight for polling; Docket Alert webhooks are the path.

Danger line No broad handroll

Foreign annuities, multi-jurisdiction chains, and litigation rules need a vendor or attorney engine.

Reliability Heartbeat required

A reminder job that silently stops is a docketing failure. The system must prove it ran.
