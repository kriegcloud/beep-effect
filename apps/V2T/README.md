# @beep/V2T

Local-first V2T desktop workspace shell

## Installation

```bash
bun add @beep/V2T
```

## Usage

```ts
import { VERSION } from "@beep/V2T"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
