# @beep/v2t

Local-first V2T desktop workspace shell

## Installation

```bash
bun add @beep/v2t
```

## Usage

```ts
import { VERSION } from "@beep/v2t"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
