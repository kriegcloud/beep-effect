# @beep/codedank-web

Next.js 16 canary app wired to the shared `@beep/ui` shadcn/Tailwind setup.

## Development

```bash
# Shared portless URL
bun run dev

# Raw Next dev server
bun run dev:raw

# Turbopack production build
bun run build

# Webpack PWA build
bun run build:pwa
```

## Dev URL

`bun run dev` serves the app at `https://codedank-web.localhost:1355`.
