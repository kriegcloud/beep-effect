# Installation

## Requirements

- Node.js 22 or later
- pnpm (recommended) or npm

## Install globally

```bash
# Using pnpm
pnpm add -g lnai

# Using npm
npm install -g lnai
```

## Install as dev dependency

```bash
# Using pnpm
pnpm add -D lnai

# Using npm
npm install -D lnai
```

## Verify installation

```bash
lnai --version
```
