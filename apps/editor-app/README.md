# @beep/editor-app

Local-first desktop editor workspace shell

## Installation

```bash
bun add @beep/editor-app
```

## Usage

```ts
import { VERSION } from "@beep/editor-app"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
