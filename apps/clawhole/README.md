# @beep/clawhole

An experimental playground for creating an agentic harness

## Installation

```bash
bun add @beep/clawhole
```

## Usage

```ts
import { VERSION } from "@beep/clawhole"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
