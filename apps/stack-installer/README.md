# `@beep/stack-installer`

Tauri and React shell for the Stack Installer P1 proof flow.

P1A proved the dry-run app surface, slice-owned verb contracts, deterministic
manifest snapshot, and UI baseline. P1 live now adds Manual Mode validators,
the Tauri proof command, and CLI artifact capture/audit for fresh-machine proof
runs. P1D now begins the first app-first live mutation lane with a focused Bun
repair flow on Linux. It still does not provision Discord, accept plaintext
credentials, implement AI Mode, or persist wider host configuration.
