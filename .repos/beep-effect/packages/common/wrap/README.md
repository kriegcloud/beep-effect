# @beep/wrap

System-wide wrapper utilities for the Effect-first monorepo.

## Installation

```bash
# This package is internal to the monorepo
# Add as a dependency in your package.json:
"@beep/wrap": "workspace:*"
```

## Development

```bash
# Type check
bun run --filter @beep/wrap check

# Lint
bun run --filter @beep/wrap lint

# Build
bun run --filter @beep/wrap build

# Run tests
bun run --filter @beep/wrap test
```
