## Features

<CardGrid stagger>
  <Card title="Unified Configuration" icon="document">
    Define your AI tool settings once in `.ai/` and sync to all supported tools.
  </Card>
  <Card title="Multiple Tools" icon="puzzle">
    Support for Claude Code, Codex, Cursor, Gemini CLI, OpenCode, Windsurf, and
    GitHub Copilot.
  </Card>
  <Card title="Type-Safe" icon="approve-check">
    Full TypeScript support with Zod schema validation.
  </Card>
  <Card title="Extensible" icon="add-document">
    Plugin architecture makes it easy to add support for new tools.
  </Card>
</CardGrid>

## Quick Example

```bash
# Initialize unified config
lnai init

# Sync to all tools
lnai sync
```

```text
claudeCode:
  + .claude/CLAUDE.md
  + .claude/settings.json

codex:
  + .codex/config.toml
  + .codex/skills/

cursor:
  + AGENTS.md
  + .cursor/mcp.json

opencode:
  + AGENTS.md
  + opencode.json

geminiCli:
  + .gemini/GEMINI.md
  + .gemini/settings.json

windsurf:
  + AGENTS.md
  + .windsurf/rules/
```
