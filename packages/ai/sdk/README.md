# @beep/ai-sdk

The shared agent sdk library for coding harness abstractions

## Installation

```bash
bun add @beep/ai-sdk
```

## Usage

```ts
import { VERSION } from "@beep/ai-sdk"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
