# @beep/editor-lexical

Lexical adapter layer for the canonical editor document model

## Installation

```bash
bun add @beep/editor-lexical
```

## Usage

```ts
import { VERSION } from "@beep/editor-lexical"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
