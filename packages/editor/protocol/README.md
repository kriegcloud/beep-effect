# @beep/editor-protocol

Public protocol for the local editor sidecar runtime

## Installation

```bash
bun add @beep/editor-protocol
```

## Usage

```ts
import { VERSION } from "@beep/editor-protocol"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
