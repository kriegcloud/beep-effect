# @beep/editor-client

Typed client boundary for the local editor sidecar runtime

## Installation

```bash
bun add @beep/editor-client
```

## Usage

```ts
import { VERSION } from "@beep/editor-client"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
