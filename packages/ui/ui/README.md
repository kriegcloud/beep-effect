# @beep/ui

Shared UI component library with shadcn, Tailwind v4 theme, and Storybook

## Installation

```bash
bun add @beep/ui
```

## Usage

```ts
import { VERSION } from "@beep/ui"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
