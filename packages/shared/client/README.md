# @beep/shared-client

The client package for the shared slice

## Installation

```bash
bun add @beep/shared-client
```

## Usage

```ts
import { VERSION } from "@beep/shared-client"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
