# @beep/shared-ui

The ui package for the shared slice

## Installation

```bash
bun add @beep/shared-ui
```

## Usage

```ts
import { VERSION } from "@beep/shared-ui"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
