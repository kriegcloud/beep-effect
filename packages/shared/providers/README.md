# @beep/shared-providers

An library for housing vendor services

## Installation

```bash
bun add @beep/shared-providers
```

## Usage

```ts
import { VERSION } from "@beep/shared-providers"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
