# @beep/firecrawl

Firecrawl provider package

## Installation

```bash
bun add @beep/firecrawl
```

## Usage

```ts
import { VERSION } from "@beep/firecrawl"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
