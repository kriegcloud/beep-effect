# @beep/shared-domain

The shared kernel for cross cutting domain dependencies

## Installation

```bash
bun add @beep/domain
```

## Usage

```ts
import { VERSION } from "@beep/domain"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
