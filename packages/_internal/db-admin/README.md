# @beep/db-admin

The internal db-admin for migrations

## Installation

```bash
bun add @beep/db-admin
```

## Usage

```ts
import { VERSION } from "@beep/db-admin"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
