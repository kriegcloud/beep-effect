# @beep/iam-ui

The ui package for the iam slice

## Installation

```bash
bun add @beep/iam-ui
```

## Usage

```ts
import { VERSION } from "@beep/iam-ui"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
