# @beep/iam-server

The server package for the iam slice

## Installation

```bash
bun add @beep/iam-server
```

## Usage

```ts
import { VERSION } from "@beep/iam-server"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
