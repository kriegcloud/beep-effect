# @beep/iam-domain

The domain package for the iam slice

## Installation

```bash
bun add @beep/iam-domain
```

## Usage

```ts
import { VERSION } from "@beep/iam-domain"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
