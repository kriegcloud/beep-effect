# @beep/canvas-domain

Architecture-lab domain package for the synthetic CanvasProject aggregate.

## Installation

```bash
bun add @beep/canvas-domain
```

## Usage

```ts
import { VERSION } from "@beep/canvas-domain"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Integration test
bun run test:integration

# Lint
bun run lint
```

Keep package tests aligned with the repo policy/laws guidance for integration placement and `@beep/*` package imports.
