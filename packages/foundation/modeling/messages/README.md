# @beep/messages

A small i18n/message helper package for schema issue formatting.

## Installation

```bash
bun add @beep/messages
```

## Usage

```ts
import { getLogIssues, logIssues, t } from "@beep/messages";
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
