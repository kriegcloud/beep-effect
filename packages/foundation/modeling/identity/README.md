# @beep/identity

Effect v4 identity package

## Installation

```bash
bun add @beep/identity
```

## Usage

```ts
import { VERSION } from "@beep/identity"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
