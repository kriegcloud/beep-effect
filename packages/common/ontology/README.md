# @beep/ontology

The core ontology system

## Installation

```bash
bun add @beep/ontology
```

## Usage

```ts
import { VERSION } from "@beep/ontology"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
