# @beep/data

A library containing data values & utilities

## Installation

```bash
bun add @beep/data
```

## Usage

```ts
import { VERSION } from "@beep/data"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
