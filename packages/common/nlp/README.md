# @beep/nlp

An library containing utilities, schemas & services for natural language processing

## Installation

```bash
bun add @beep/nlp
```

## Usage

```ts
import { VERSION } from "@beep/nlp"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
