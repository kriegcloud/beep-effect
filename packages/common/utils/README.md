# @beep/utils

Shared runtime utilities for beep

## Installation

```bash
bun add @beep/utils
```

## Usage

```ts
import { VERSION } from "@beep/utils"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
