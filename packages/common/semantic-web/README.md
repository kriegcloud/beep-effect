# @beep/semantic-web

An library containing utilities, schemas & services for working with the semantic web

## Installation

```bash
bun add @beep/semantic-web
```

## Usage

```ts
import { VERSION } from "@beep/semantic-web"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
