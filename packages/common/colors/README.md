# @beep/colors

terminal output formatting with ANSI colors.

## Installation

```bash
bun add @beep/colors
```

## Usage

```ts
import { VERSION } from "@beep/colors"
```

## Development

```bash
# Build
bun run build

# Type check
bun run check

# Test
bun run test

# Lint
bun run lint:fix
```
