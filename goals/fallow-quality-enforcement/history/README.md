# Fallow Quality Enforcement History

Store round summaries, critic findings, implementation evidence, and closeout
artifacts here.

Required machine-readable file:

- `history/review-rounds.jsonc`

Future supporting closeout files:

- `history/round-<n>-critics.md`
- `history/round-<n>-fixes.md`
- `history/outputs/p0-research-closeout.md`
- `history/outputs/implementation-closeout.md`

`review-rounds.jsonc` is the machine-readable source for required critic
findings, closure status, waivers, and acceptance commands. Markdown round
files are supporting narrative only.
