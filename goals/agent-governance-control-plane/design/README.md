# Design Docs

This directory holds the durable design record for the agent governance control plane.

## Design Set

- [foundation-and-law-canon.md](./foundation-and-law-canon.md) — source-of-truth order, law families, and exception policy
- [agent-topology-and-role-contracts.md](./agent-topology-and-role-contracts.md) — canonical agent catalog, ownership, and blocker contracts
- [workflow-lifecycle-and-phase-gates.md](./workflow-lifecycle-and-phase-gates.md) — universal category loop and gating rules
- [artifact-contracts-and-prompt-assets.md](./artifact-contracts-and-prompt-assets.md) — packet definitions and reusable prompt surfaces
- [enforcement-and-verification-contract.md](./enforcement-and-verification-contract.md) — law coverage mapping, command policy, and verification expectations
- [adoption-and-consumer-integration.md](./adoption-and-consumer-integration.md) — consumer inheritance model and bootstrap workflow

## Design Rule

These docs are required to stay generic and repo-wide. Downstream initiative details belong in the downstream initiative, not in this packet.
