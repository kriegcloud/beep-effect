# adjunct → Effect v4 Port Map

> Source: **adjunct** (`@template/basic`, Effect `^3.17.7`) → target **`@beep/nlp`** (Effect v4, foundation/capability).
> All v4 targets verified against `.repos/effect-v4` source + `.repos/effect-v4/migration/v3-to-v4.md` (Import Map L13, No-Counterpart L311, API Renames L362).
> Repo-law overlay (CLAUDE.md): no `as`/`any`/`!`, no native `Map`/`Set`/`Array`/`Error`/`null`/`JSON.parse`, `Clock` over `Date.now`, `Context.Service` class, `Effect.fn`/`Effect.fnUntraced` over bare `Effect.gen`, `$NlpId` identity + `$I.annote` on every schema/error, `TaggedErrorClass` from `@beep/schema`.

## (a) Overview + Cross-Cutting Rename Checklist

The adjunct port collapses three v3 dependency families into v4 core (`effect`, `effect/unstable/*`, `effect/testing`), drops two presentation-only packages with no v4 home, replaces one (`zod`), and re-expresses the `@effect/typeclass` algebra layer onto `effect/Combiner` + `effect/Reducer`. ## (e) Proofs / Laws to Preserve (full categorical fidelity)

All laws are JSDoc prose / property tests in adjunct; the v4 port must realize them as `effect/testing/FastCheck` properties. Arbitraries live in `test/arbitraries.ts`.

### Adjunction (Free ⊣ Forgetful) — `test/Adjunction.test.ts` (+ `Extended.verifyAdjunction`)
- **Right triangle (sentencize):** `G(ε)∘η_G = id_G` — `text→sentencize→join→sentencize ≈ sentencize` (up to whitespace/punct normalization). Arbitraries: `nonEmptyText`, `shortText`; `numRuns=100`.
- **Left triangle (sentencize):** `ε_F∘F(η) = id_F` — `sentences→join→sentencize ≈ sentences`. Arbitrary: `sentenceArray` (deliberately excludes `[.!?]`); `numRuns=100`.
- **Right triangle (tokenize):** `text→tokenize→join→tokenize ≈ tokenize`. Arbitrary: `nonEmptyText`; `numRuns=100`.
- **Left triangle (tokenize):** `tokens→join→tokenize ≈ tokens`. Arbitrary: `tokenArray`; `numRuns=100`.
- **Composed adjunction (sentencize∘tokenize):** forward then backward yields valid non-empty sentences. `numRuns=50`.
- **Naturality:** `transform∘sentencize = sentencize∘(map transform)` (uppercase + filtering). `numRuns=50`.
- **Graph structure:** singleton + sentence-children reconstructs original via `simpleJoin` (normalize equality).
- **Edge cases:** empty/single-word/no-boundary/multi-punctuation handled gracefully.
- **Adjunction unit/counit duality:** `Free (expand, one-to-many) ⊣ Forgetful (aggregate, many-to-one)` via `makeAdjunction` (`TypeClass.ts`); forgetful side = monoid fold.

### Monoid laws — `test/Algebra/{Monoid,NLPMonoid,NLPMonoids}.test.ts` (harness `testMonoidLaws`/`checkLaws`)
- **Left identity:** `empty ⊕ x = x`. **Right identity:** `x ⊕ empty = x`. **Associativity:** `(x⊕y)⊕z = x⊕(y⊕z)`.
- **`combineAll`:** `combineAll([x,y,z]) = x⊕y⊕z`; `combineAll([]) = empty` (NLPMonoids, via typeclass `Monoid.combineAll`).
- **`combineMany`:** `combineMany(x,[y,z]) = x⊕y⊕z`; `combineMany(self,[]) = self` (NLPMonoids).
- **Lawful concrete instances:** `StringConcat/Join(sep)`, `NumberSum/Product (bounded)/Max/Min (bounded)`, `ArrayConcat`, `BooleanAll/Any`, `Product`, `Endo` (equality by sampling `[0,1,-1,42,100]`), `VectorAdd(3)`, `Dual`. Arbitraries: `fc.string/integer({min,max})/array/boolean/tuple/func`.
- **Lawful NLP instances (flat + nested twins):** `Token(s).Concat/BagOfWords/SetUnion`, `Sentence(s).Concat/Array`, `Document(s).Text/Stats`, `NamedEntity(List)/Linguistic.NamedEntities`, `DependencyParse`, `TermFrequency`, `DocumentFrequency`, `Vocabulary`, `WeightedTokens`, `NGramFrequency`, `TextAnalysis(Monoid)/Composite.TextAnalysis`. Arbitraries: `bagOfWords`, `namedEntity`, `dependencyEdge`, `documentStats`, `weightedTokens`, `ngramFrequency`, `fc.double({noNaN,noDefaultInfinity})`.
- **Commutativity:** `BagOfWords` `bow1⊕bow2 = bow2⊕bow1`; `SetUnion` `set1∪set2 = set2∪set1`.
- **Idempotence:** `SetUnion` `set∪set = set`.
- **Non-negativity + correct summation:** `DocumentStats` (wordCount/sentenceCount/charCount add).
- **Monoid HOMOMORPHISM (bag-of-words):** `bow(xs++ys) = bow(xs) ⊕ bow(ys)` — `aggregateTokens` is a homomorphism `(List,++)→(BagOfWords,⊕)`.
- **Monoid HOMOMORPHISM (vocabulary):** `vocab(xs++ys) = vocab(xs) ∪ vocab(ys)`.
- **TF normalization law:** `bagOfWordsToTF` → `Σ tf = 1.0` (±0.0001) for non-empty; empty→empty.
- **TF-IDF correctness:** `computeTFIDF(tf,df,N) = tf·log(N/df)`; higher TF dominates rarity; TF-absent terms excluded.

### Functor / Applicative / Monad / Foldable / Traversable / Alternative — `TypeClass.ts`, `GraphOps.ts`, `fixes.test.ts`
- **Functor identity & composition:** `map(id)=id`, `map(f∘g)=map(f)∘map(g)` (`GraphOps.mapNodes`, `EffectGraph.map`, `TG.mapNodes`).
- **Bifunctor:** `bimap = mapNodes ∘ mapEdges`.
- **Applicative (parallel composition):** `ap` = Cartesian product of fn-nodes × value-nodes; `product = Effect.all/zip` semantics.
- **Monad:** left identity `flatMap(pure(a),f)=f(a)`; right identity `flatMap(m,pure)=m`; associativity `flatMap(flatMap(m,f),g)=flatMap(m,x=>flatMap(f(x),g))`.
- **Foldable:** `fold(f,id,[])=id`; `fold(f,id,[x])=f(id,x)`; `fold(f,id,xs++ys)=f(fold xs, fold ys)`.
- **Alternative:** `empty` is identity for `alt`.

### Graph laws — `EffectGraph.ts`, `TextGraph.ts`, `GraphOps.ts`, `fixes.test.ts`, `TextGraph.test.ts`
- **Catamorphism (`cata`):** bottom-up fold over the graph functor's initial algebra; children-before-parents (topological), memoized; homomorphism to any `GraphAlgebra<A,B>` (char count + bottom-up text collection). Demoed in `Program.ts`/`AdvancedDemo.ts`.
- **Anamorphism (`ana`):** dual of `cata`; unfold from seed via `GraphCoalgebra` → `(data, childSeeds)`; `cata/ana` form the hylomorphism pair.
- **Adjoint-functor SEARCH:** `buildIndex (index functor) ⊣ queryIndex/queryIndexUnion (query functor)`; round-trip `query(buildIndex(g))` correct membership + union.
- **DAG / acyclicity invariant:** every constructed graph satisfies `Graph.isAcyclic`; `TextGraph.addChildren` rejects cycle-creating edges.
- **Idempotence:** `tokenizeNodes`, `addPOS/Lemma/EntityAnnotations` — `f(f(g)) = f(g)`.
- **Traversal:** DFS/BFS over `Graph.values` produce expected sequences/counts.

### Annotation laws — `test/AnnotatedTextGraph.laws.test.ts` (over `WinkBackend`)
- **Idempotence:** `addPOS/Lemma/EntityAnnotations` twice = once (node-count stable).
- **Preservation:** each op preserves text-node count; graph stays valid/acyclic after POS+lemma+entity.
- **Correctness:** POS nodes have non-empty text+tag+numeric position; lemma nodes non-empty token+lemma+position; entity nodes valid spans (`start<end`, within `[0,len]`).
- **Count law:** `|POSNodes| = |tokenNodes|`, `|lemmaNodes| = |tokenNodes|`.
- **Composition / order-independence:** POS-then-lemma vs lemma-then-POS equal per-type counts; incremental == single fully-annotated. Arbitraries: `fc.stringMatching`, `fc.array`, `fc.constantFrom`.

### NLP operation laws — `test/NLPOperations.laws.test.ts`, `NLPService.linguistic.test.ts`, `fixes.test.ts`
- **Idempotence:** `normalizeWhitespace`, `removePunctuation`, `removeExtraSpaces`, `stem`, `removeStopWords` (and lemma idempotence on lemma strings).
- **Monotonicity:** `|removeStopWords(tokens)| ≤ |tokens|`; `|removePunctuation(text)| ≤ |text|`.
- **Preservation:** `wordCount(text) = |tokenize(text)|`; `Σ bagOfWords.values = |tokens|`; `bagOfWords.keys ⊆ tokens`; POS/lemma token-count preservation.
- **String-similarity metric-space:** reflexive `sim(x,x)=1` (exact `===1.0`); symmetric `sim(x,y)=sim(y,x)`; bounded `0≤sim≤1` (Jaro-Winkler; cross-ref `fixes.test.ts` Jaro regression). Tolerance `0.0001`.
- **Composition:** `tokenize∘normalize` vocab equality; punctuation-removal token-reconstruction identity; `removeStopWords>>stem` order-independence (SKIPPED — must remain `it.skip`).
- **N-gram counting:** `|ngrams(text,n)| = max(0, |tokens|-n+1)`; `ngrams(text,1) = tokens`.
- **Sentence-boundary preservation:** re-sentencizing joined sentences preserves count ±1.

### Backend / Executor / Pipeline laws — `Backends/Composition.test.ts`, `GraphOperations/Executor.test.ts`, `Serialization.ts`, `CorpusOps.ts`
- **Backend capability/union:** `FallbackBackend` reports union of capabilities, falls back on failure/`BackendNotSupported`; `CachingBackend` preserves name+capabilities, returns identical cached results. Backend composition forms a coproduct/sum (identity = no-op backend).
- **Executor metric/caching:** cache miss then hit on identical input; `nodesProcessed/nodesCreated` counts; cost scales with node count; empty graph → empty result.
- **Dependency-graph invariants (`validateDependencyGraph`):** acyclic (`Graph.isAcyclic`), valid topological order covering all nodes (`Graph.topo`), ≥1 root (`Graph.externals`), every node reachable via DFS from roots (`Graph.dfs`).
- **Serialization round-trip:** `serializePipeline → deserializePipeline` reconstructs an equivalent Pipeline (currently weakened — Map/FlatMap/ZipWith compiler handlers are stubs, `reconstructSchemaFromAST` best-effort).
- **CorpusStatistics monoid:** `emptyStatistics` identity; `combineStatistics` associative; parallel aggregation preserves the homomorphism.
- **`cosineSimilarity` ∈ [0,1]**, symmetric (upper-triangular pairwise matrix).
- **Streaming round-trips (worth FastCheck):** `parseJsonlString(stringifyJsonl(records)) == records`; `writeJsonl→readJsonl` round-trips; partition invariant `successCount + errorCount == totalLines`.
