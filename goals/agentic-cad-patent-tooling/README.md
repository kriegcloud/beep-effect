# Agentic CAD — Patent Tooling

Survey-and-integration initiative for AI/LLM-driven CAD tooling in IP/patent work, evaluated
for fit with this repo's local-first agentic professional runtime (see
[`goals/agentic-professional-runtime`](../agentic-professional-runtime/) and
[`PRD-ambient-billing-copilot.md`](../../PRD-ambient-billing-copilot.md)).

The initiative covers four jobs-to-be-done: (1) USPTO/WIPO patent **figures**, (2) invention
**3D modeling**, (3) **prior-art / reverse-modeling**, and (4) the full **disclosure→figure
pipeline**. Tools are scored on a two-axis rubric — an *integration ladder* (native MCP →
REST/gRPC API → CLI/headless → scriptable library → file-format interop) and a
*deployment/privilege* axis (cloud / hybrid / self-hostable / fully-local, with Rule 1.6
confidentiality implications) — because the production runtime mandates on-device processing
for privileged client data.
