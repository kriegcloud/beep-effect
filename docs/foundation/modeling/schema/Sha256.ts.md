# validation

## Sha256Hex

Branded schema for canonical lowercase SHA-256 hex digests (64 hex characters).

**Example**

```ts
import * as S from "effect/Schema"
import { Sha256Hex } from "@beep/schema/Sha256"

const digest = S.decodeUnknownSync(Sha256Hex)(
  "e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855"
)
console.log(digest)
```

**Signature**

```ts
declare const Sha256Hex: AnnotatedSchema<S.brand<S.String, "Sha256Hex">>
```

[Source](https://github.com/kriegcloud/beep-effect/tree/main/packages/foundation/modeling/schema/src/Sha256.ts#L57)

Since v0.0.0

## Sha256HexFromBytes

One-way schema that decodes a byte array into a canonical lowercase SHA-256
hex digest. Encoding back is intentionally forbidden.

**Example**

```ts
import { Effect } from "effect"
import * as S from "effect/Schema"
import { Sha256HexFromBytes } from "@beep/schema/Sha256"

const program = S.decodeUnknownEffect(Sha256HexFromBytes)(new Uint8Array())
const result = Effect.runPromise(program)
console.log(result)
```

**Signature**

```ts
declare const Sha256HexFromBytes: AnnotatedSchema<S.decodeTo<AnnotatedSchema<S.brand<S.String, "Sha256Hex">>, S.Uint8Array, never, never>>
```

[Source](https://github.com/kriegcloud/beep-effect/tree/main/packages/foundation/modeling/schema/src/Sha256.ts#L101)

Since v0.0.0

## Sha256HexFromHexBytes

One-way schema that decodes a hex-encoded byte string into a canonical
lowercase SHA-256 hex digest via `Sha256HexFromBytes`.

**Example**

```ts
import { Effect } from "effect"
import * as S from "effect/Schema"
import { Sha256HexFromHexBytes } from "@beep/schema/Sha256"

const program = S.decodeUnknownEffect(Sha256HexFromHexBytes)("68656c6c6f")
const result = Effect.runPromise(program)
console.log(result)
```

**Signature**

```ts
declare const Sha256HexFromHexBytes: AnnotatedSchema<S.compose<AnnotatedSchema<S.decodeTo<AnnotatedSchema<S.brand<S.String, "Sha256Hex">>, S.Uint8Array, never, never>>, S.Uint8ArrayFromHex>>
```

[Source](https://github.com/kriegcloud/beep-effect/tree/main/packages/foundation/modeling/schema/src/Sha256.ts#L144)

Since v0.0.0
