# Research Reports

Place synthesized research reports for the JSDoc quality enforcement initiative
in this directory.

## Report Template

```md
# <Research Topic>

## Question

## Scope

## Repo Evidence

## External Evidence

## Options

## Tradeoffs And Risks

## Recommendation

## Open Questions
```

## Rules

- Prefer one report per research lane.
- Keep reports synthesized and decision-oriented, not transcript-shaped.
- Link to repo files and external sources used by the research.
- Put bulky raw outputs in `../history/outputs/` only when they are useful
  provenance.
