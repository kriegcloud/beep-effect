# Agent Memory Research Spec

## Purpose

This initiative exists to gather and compare the external agent-memory
landscape so future repo work can make grounded architectural choices.

## Contract

- The documents in [research/](./research) are informative, not normative
  implementation contracts.
- This initiative should inform downstream architecture work, especially
  repo-memory and expert-memory efforts, but should not be treated as a direct
  shipping checklist.
- Any concrete implementation decision that survives beyond research should be
  restated in the downstream initiative that will own the work.
