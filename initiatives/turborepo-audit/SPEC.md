# Turborepo Audit Spec

## Purpose

This initiative captures the audit contract for evaluating the repo's
Turborepo usage, command surface, CI posture, and related tooling integration.

## Contract

- [history/decisions.md](./history/decisions.md) preserves the locked audit
  defaults and source rules.
- [history/outputs/](./history/outputs) holds the per-topic outputs and summary
  artifacts produced by the audit.
- [research/official-topic-surface-map.md](./research/official-topic-surface-map.md)
  is supporting reference material, not the main audit contract.
